# InstaDeep Ltd

Tutorial for Nvidia GTC 2020

```bash
git clone https://github.com/instadeepai/gtc-course-2020.git

Docker Instructions
- Use the Dockerfile to build a docker image using: `docker build -t instadeep-gtc .`
- Run a container from this image using:  `docker run -p 8888:8888 --runtime=nvidia instadeep-gtc`.
- Once the container is running, visit the content in your browser at: localhost:8888.
```
